# sdmx-parsers Examples

Examples demonstrating `sdmx-parsers` usage.

## Running Examples

```bash
cargo run --example <name>
```

---

## Planned Examples

| Example    | Purpose                                        |
|------------|------------------------------------------------|
| parse_csv  | Parse SDMX-CSV data format with streaming      |
| parse_json | Parse SDMX-JSON structure and observation data |
| parse_xml  | Parse SDMX-ML structure and data messages      |
