# User Guides

Comprehensive tutorials and walkthroughs for library users.

## What Are Guides?

Guides provide **step-by-step walkthroughs** of common use cases and patterns. They complement the [API reference](https://docs.rs/sdmx-rs) (rustdoc) by explaining *how* and *why*, not just *what*.

See [Using sdmx-rs](../user/README.md) for quickstart documentation.
